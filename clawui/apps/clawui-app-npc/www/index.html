📌 NPC 客户端说明

NPC (NPS Client) 是 NPS 内网穿透的客户端，用于连接到 NPS 服务器。

使用步骤：

  1. 在 NPS 服务端创建客户端，获取验证密钥 (vkey)
  2. 在上方填写服务器地址、端口和 vkey
  3. 保存配置并启动服务
  4. 在 NPS 服务端配置隧道（TCP/UDP/HTTP/HTTPS）

隧道类型说明：

  • TCP 隧道： 适用于 SSH、远程桌面等 TCP 服务
  • UDP 隧道： 适用于 DNS、游戏等 UDP 服务
  • HTTP 隧道： 适用于网站
  • HTTPS 隧道： 适用于加密网站
