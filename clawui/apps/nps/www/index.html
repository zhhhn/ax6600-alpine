Usage

NPS allows you to expose local services behind NAT to the internet:

  • HTTP/HTTPS proxy: Access web services remotely
  • TCP/UDP tunnel: Forward any TCP/UDP port
  • SOCKS5: Full proxy support
  • P2P: Direct peer-to-peer connections

Example: Expose local web server (port 80)

Server side (NPS): Create TCP tunnel
  Local port: 80
  Remote port: 8080
  
Access: http://your-server-ip:8080
